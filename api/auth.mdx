Travomatrix uses API keys for server-side authentication and a Role-Based Access Control (RBAC) system for agency users.

## Roles

- `SUPER_ADMIN`: Full system access.
- `AGENCY_ADMIN`: Manage a specific agency's users and finances.
- `AGENT`: Standard search and booking capabilities.
- `FINANCE`: Manage wallets and view transaction history.

## User Login

`POST /api/v1/agency/login`

Authenticate a user and retrieve their profile and agent context.
