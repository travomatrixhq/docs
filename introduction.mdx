Travomatrix is a high-performance flight booking engine designed for B2B travel platforms. It provides a robust suite of APIs for agency management, prepaid wallet systems, and seamless flight operations across multiple GDS and NDC providers.

## Key B2B Features

- **Multi-tenant Agency Management**: Manage multiple travel agencies, users, and roles.
- **Prepaid Wallet & Credit Ledger**: Strict financial control with atomic wallet operations.
- **B2B Optimized Search**: Real-time commission calculation and net fare display.
- **Asynchronous Booking**: Background processing for high-volume booking environments.
- **Post-Booking Lifecycle**: Full support for cancellations, refunds, and rescheduling.
